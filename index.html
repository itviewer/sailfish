解锁前将手机Android系统恢复出厂设置，务必将Android升级到最新系统（7.1.1，大于228版本，不要使用8），否则刷机失败
本刷机包为官方刷机包，非社区版，使用Android需要自己帐号已经购买jolla服务。

准备

不解锁bootloader不能刷机，所以需要先解锁bootloader，解锁bootloader需要获取解锁代码。

允许手机解锁

  • 手机进入Android，设置-关于手机-软件版本，点多次，直到提示已经进入开发者模式，回到设置—开发者选项——勾选 开启oem解锁

查看是否可以解锁/是否已经解锁

  • 拨号界面输入 *#*#7378423#*#*
  • 依次点击菜单 Service Info——Configuration，滑到最后，查看Rooting Status
  • 如果为 Bootloader unlocked： Yes 则已经解锁，如果为 Bootloader unlock allowed： Yes 则为未解锁（可解锁）

获取unlock code

需要参考官方指导，https://developer.sonymobile.com/unlockbootloader/unlock-yourboot-loader/

  1. 在 官方网址 选择“Xperia X”后点击“Continue”
  2. 在 email 验证界面输入你的email，勾选“Terms & conditions” 和 “I'm not a robot” 后点 “submit”，然后在收到的确认邮件中点击“Click here to proceed”
  3. 在拨号界面输入 *#06#，记下IMEI ，在上述打开的地址输入IMEI，勾选下面两个选择框，然后“submit”
  4. 记下上面得到的 unlock code

刷机或解锁环境准备

安装刷机工具

如果使用包含sailfish镜像的一键刷机包，则里面已经包含fastboot工具，不需要单独下载。

若想单独安装刷机工具，点击这里下载 。下载后解压，解压目录最好不要含中文和空格

使用方法：进入解压后的目录，按住shift并右键，点击“在此处打开命令窗口”

驱动安装

若驱动没有正确安装，则刷机时fastboot找不到设备。

重启电脑，临时禁用强制驱动签名

此操作仅当安装驱动时执行，驱动安装成功后不需要此步骤

具体操作参考 https://zhidao.baidu.com/question/2118293227537379627.html

安装驱动

  • 手机关机，按住音量+键同时将数据线插入电脑USB接口，待手机指示灯变为蓝色，手机进入fastboot模式
  • 电脑会自动安装驱动，但会失败，忽略即可
  • 下载 驱动 并解压
  • 右击“计算机”图标（win7，若win10则为此电脑），选择“管理”—“设备管理”—“其它设备”—“S1Boot Fastboot”
  • 选择“更新驱动程序软件”—“浏览计算机以查找驱动程序软件”—“从计算机的设备驱动程序列表中选择”—“下一步”—“从磁盘安装”—“浏览”找到刚才解压的驱动程序目录，选择“android_winusb.inf”—“确定”
  • 点击第二个“Android bootloader interface”—“下一步”—“是”—“始终安装此驱动程序软件”

解锁bootloader

如果手机已经解锁忽略该操作。
解锁bootloader会重置系统为出厂状态丢失所有数据

解锁bootloader

若手机已经在fastboot模式，直接下一步，若手机不是在fastboot模式

  • 关掉手机，按住音量加键同时将数据线插入电脑USB接口，待指示灯显示为蓝色，手机进入fastboot模式

若已经进入fastboot模式则，

  1. 在刷机工具目录打开cmd窗口，输入 fastboot devices 查看是否检测到设备，若显示类似BH90063H5K fastboot，则驱动正常，若没有内容则驱动安装失败，重新检查上面步骤
  2. 执行 fastboot -i 0x0fce oem unlock 0xNNNNNNNNN 解锁（将NNNNNNNNN替换为前面得到的unlock code）

解锁后第一次重启会提示一句英文，大概意思是“你的设备已经解锁，设备不受信任”然后5秒后自动重启，重启后系统为恢复出厂设置状态，需要等待一会才能进入系统界面，以后每次重启也都会上述提示。

刷官方系统

按照上面步骤，确认已解锁bootloader并能用fastboot devices看到手机后

  • 这里下载一键刷机包后解压
  • 双击flash-on-windows.bat，回车，稍等片刻提示完成，断开数据线，重启手机

恢复出厂设置

移除sailfish，恢复Android系统
下载官方恢复Android工具emma
具体使用参考 https://developer.sonymobile.com/open-devices/flash-tool/use-the-flash-tool-for-xperia-devices/

  • 工具安装完启动时需要联网，可能需要翻，墙才能正常运行
  • 关机，按音量减键同时插入电脑USB接口，指示灯变绿，自动安装驱动后识别手机
  • 选择 大于 34.3 的服务版本，点应用服务
  • 软件会下载2.5G的东西，然后将手机恢复出厂设置
  • 断开usb，启动手机
