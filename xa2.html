本教程适用于XA2系列

解锁前最好确保Android版本为8.1
使用Android需要自己帐号已经购买jolla服务。本刷机包可能版本较低，可刷机后通过在线方式升级

准备

不解锁bootloader不能刷机，所以需要先解锁bootloader，解锁bootloader需要获取解锁码（unlock code）。

查看是否可以解锁/是否已经解锁

  • 拨号界面输入 *#*#7378423#*#*
  • 依次点击菜单 Service Info——Configuration，滑到最后，查看Rooting Status
  • 如果为 Bootloader unlocked： Yes 则已经解锁，如果为 Bootloader unlock allowed： Yes 则为未解锁（可解锁）
  • 如果已经解锁可以跳过相关步骤，否则按下面步骤继续

允许手机解锁

  • 手机进入Android，设置-[系统]-关于手机-构建版本（Build number），点多次，直到提示已经进入开发者模式，回到设置-[系统]-开发者选项
  • 勾选开启oem解锁
  • 确认 设置-[系统]-开发者选项 中 USB调试（USB debugging）已经勾选

获取unlock code

  1. 在拨号界面输入 *#06#，记下IMEI，双卡可能有两个IMEI，记下第一个
  2. 在 官方网址 拉到最下面，选择相应版本，如"Xperia XA2"，"Xperia XA2 Plus"，然后再下拉网页
  3. 在 “Enter IMEI, IDID or MEID” 输入框输入你的IMEI，勾选第一项和第二项，点击Submit，输入验证码，点击Continue
  4. 上述步骤最好有个VPN，不然可能会出现The captcha could not be verified 提示
  5. 如果上面通过，会给出unlock code，记下上面得到的 unlock code

下载刷机包（安装刷机工具）

驱动安装

如果电脑能够自动安装驱动，可跳过此步骤，否则需要手动安装驱动

临时禁用强制驱动签名

具体操作参考

安装驱动

  • 手机关机，按住音量加键同时将数据线插入电脑USB接口，待手机指示灯变为蓝色（如果灯变绿，重新关机，断开数据线，重试），手机进入fastboot模式
  • 电脑会自动安装驱动，但会失败，忽略即可
  • 下载 驱动 并解压
  • 右击“计算机”图标（win7，若win10则为此电脑），选择“管理”—“设备管理”—“其它设备”—“Android”
  • 选择“更新驱动程序软件”—“浏览计算机以查找驱动程序软件”—“从计算机的设备驱动程序列表中选择”—“下一步”—“从磁盘安装”—“浏览”找到刚才解压的驱动程序目录，选择“sa0114adb.inf”—“确定”
  • 点击“Sony sa0114 ADB Interface Driver”—“下一步”—“是”—“始终安装此驱动程序软件”

解锁bootloader

如果手机已经解锁忽略该操作。
解锁bootloader会重置系统为出厂状态丢失所有数据

解锁bootloader

若手机已经在fastboot模式，直接下一步，若手机不是在fastboot模式

  • 关掉手机，按住音量加键同时将数据线插入电脑USB接口，待指示灯显示为蓝色，手机进入fastboot模式

若已经进入fastboot模式则，

  1. 在刷机包目录打开cmd窗口（如何打开cmd窗口并进入指定目录，请百度，win10和win7可能有区别）
  2. 在cmd窗口下，输入 fastboot devices（或者.\fastboot.exe devices）后回车，查看是否检测到设备，若显示类似BH90063H5K fastboot，则驱动正常，若没有内容则驱动安装失败，重新检查上面 驱动安装 步骤
  3. 执行 fastboot -i 0x0fce oem unlock 0xNNNNNNNNN （或者 .\fastboot.exe -i 0x0fce oem unlock 0xNNNNNNNNN） 解锁（将NNNNNNNNN替换为前面得到的unlock code）

解锁后第一次重启手机会提示一句英文，大概意思是“你的设备已经解锁，设备不受信任”然后5秒后自动重启，重启后系统为恢复出厂设置状态，需要等待一会才能进入系统界面，以后每次重启也都会上述提示。

刷官方系统

按照上面步骤，确认已解锁bootloader并能用fastboot devices命令看到手机后，在刷机包目录

  • 双击flash-on-windows.bat，回车，稍等片刻提示完成，断开数据线，重启手机
